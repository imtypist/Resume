Hidden Camera Detection

Motivated by more and more hotel candid shots events, we try to detect if there exists any hidden camera through reading EMI signals from the power line, which is inspired by ElectriSense (Ubicomp'10). We built a PLI module and used USRP N210 to construct the experiment platform.

BlockSense: Blockchain + Crowdsensing [video]

We combine blockchain and crowdsensing to build a trustworthy decentralized crowdsensing platform, which breaks the traditional centralized triangular architecture. We leverage miners to verifying data quality while without knowing the content of data.

Map Attacker

We observe the fact that navigation apps will collect users' GPS data to calculate the real time traffic status. The main idea of this project is to utilize this fact to defraud navigation apps by uploading fake GPS data.
